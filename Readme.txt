【如何启动】
双击index.html
最佳分辨率:1920*1080，如果画面显示不全，请调低浏览器缩放
建议F11全屏游戏

【操控方式】
A：向左移动
D：向右移动
E：交互
Space：跳跃
Enter：下一条对话
Shift：冲刺
Esc：暂停菜单

键位可在设置界面调整
特别地，对于小游戏俄罗斯方块部分，强烈建议改成右手方向键移动左手z，x，c旋转和暂存的方式

【结局触发条件】
坏结局：在主线平台跳跃关卡中，坠落进虚空触发，只会触发一次
普通结局：一般地，完成主线即可触发
真结局：要求当前账号拥有成就【贪吃蛇？】【再接再厉】，不需要成就在同一游戏流程中完成；并且在单次游戏流程中，从未触发过坏结局

完成真结局后，将播放"./ed/staff.mp4"视频
